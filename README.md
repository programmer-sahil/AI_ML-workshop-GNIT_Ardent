## 🔍 Project Details

### 1. Student Intro Generator
Generates a formatted student introduction card from user input fields like name, age, course, and hobbies.
- **Skills:** Python basics, string formatting, Tkinter GUI

---

### 2. Smart Grade Calculator
Calculates grades based on marks input, applies grade boundaries, and displays pass/fail status.
- **Skills:** Conditional logic, loops, Tkinter GUI

---

### 3. Simple Calculator App
A functional desktop calculator with basic arithmetic operations built using Tkinter.
- **Skills:** Event handling, Tkinter layout, Python operators

---

### 4. Contact Book Mini Project
A CLI-based contact manager with add, search, update, and delete functionality using file storage.
- **Skills:** File I/O, dictionaries, functions, CRUD operations

---

### 5. Student Performance Data Analyzer
Analyzes a dataset of student marks — computes averages, top performers, subject-wise trends, and visualizes results.
- **Skills:** Pandas, Matplotlib, data cleaning, statistical analysis

---

### 6. Sales and Performance Dashboard
Reads sales data and generates visual reports — bar charts, line graphs, and KPI summaries.
- **Skills:** Pandas, Matplotlib, data aggregation, dashboard design

---

### 7. House Price Prediction Mini Model
Predicts house prices using regression on features like area, rooms, and location.
- **Skills:** Scikit-learn, Linear Regression, data preprocessing, model evaluation

---

### 8. Spam Message Detector
Classifies SMS/email messages as spam or not spam using NLP and a Naive Bayes classifier.
- **Skills:** Scikit-learn, TF-IDF, Naive Bayes, text preprocessing, NLP
